## 🚀 第一章 项目概述

### 1.1 项目简介

**智慧数据可视化平台**是一款基于 Spring Boot + Angular 的企业级数据可视化管理系统。平台集成了 AI 智能分析、数据可视化、权限管理等功能，为企业提供一站式的数据管理与分析解决方案。

**项目特点**：
- 🤖 **AI 智能助手**：支持自然语言交互，智能分析数据趋势
- 📊 **丰富图表**：多种图表类型，实时展示数据变化
- 🔐 **权限管理**：完善的角色权限体系，保障数据安全
- 💾 **数据持久化**：聊天记录自动保存，数据永不丢失
- 📥 **批量导入导出**：支持 Excel 批量操作，提升工作效率

#### 2.1.3 技术实现

- **后端**：`AuthController` 处理登录请求，`AuthService` 验证用户信息
- **安全**：使用 HTTP Session 管理会话

#### 2.1.4 登录流程
```
用户输入账号密码
    ↓
前端发送登录请求到后端
    ↓
后端验证用户名密码
    ↓
验证成功：返回用户信息，建立会话
验证失败：返回错误信息
    ↓
前端跳转至主页面或显示错误提示
```
